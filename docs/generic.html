This program uses RSA to encrypt the AES key files (Key and IV) with the receiverâ€™s public key. Before the AES key files are encrypted, they are used to encrypt your data with AES256. The encrypted data can be sent with the encrypted key files to the other party and can be decrypted with the RSA private key of the receiver. These decrypted keys can then be used to decrypt the encrypted data with CryptoAlgo.
