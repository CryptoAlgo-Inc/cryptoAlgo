3 Simple Steps

1. Key in text

Simpily key in the text that you wish to encrypt, and copy the encrypted text.

2. Encrypt the AES keyfiles

Encrypt the AES keyfiles with the other party's RSA public key with the inbuilt, easy to use tools.

3. Send the encrypted output

Send the encrypted text produced in step 1 with the encrypted AES keyfiles to the other party.
